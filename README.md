# Tanuki-VS-Toad
## Overview
2D Game made in Unreal Engine 5 with Blueprints

Gameplay Video: https://youtu.be/D_AFblE8Zjo


## Link

## Description

You playing as a King.

Your goal is to get to the end of the Castle.

You can use A and D to move and SpaceBar to Jump.

You can jump on top of mob heads to defeat them, just like in one game with plumberer.
